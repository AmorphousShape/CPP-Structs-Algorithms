\subsubsection*{template$<$typename key\+Type, typename value\+Type$>$\newline
class RBNode$<$ key\+Type, value\+Type $>$}
Node for a Red-\/\+Black Tree. 

This class represents a node in a Red-\/\+Black Tree. It contains pointers to its left and right children, its parent, and its key and value. It also contains the color of the node (Red or Black) and the size of the subtree rooted at this node.

This class also has all of its elements set to public. This is because it is not intended to be used on its own; rather, it is for use by the \doxylink{class_r_b_tree}{RBTree} class, which requires direct access to the node\textquotesingle{}s elements to avoid unnecessary function calls.
